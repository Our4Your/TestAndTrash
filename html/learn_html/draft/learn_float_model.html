Черновик

Свойство clear

Значение Описание
left Плавающие элементы не разрешены с левой стороны. Т.е. поток будет опущен(переведён) вниз на ширину блока в котором установлено свойство clear: left Но при условии что слева будет плавающий элемент ЕСЛИ будет родитель то ничено не произойдёт!
right Плавающие элементы не разрешены с правой стороны. Т.е. поток будет опущен(переведён) вниз на ширину блока в котором установлено свойство clear: right Но при условии что справа будет плавающий элемент ЕСЛИ будет родитель то ничего не произойдёт!
both Плавающие элементы не разрешены с обеих сторон. Прерыватель потока!! Поток будет опущен вниз на ширину самого широкого блока в "в верхлежащем потоке" Это свойство можно применить и к не-float блокам(при этом float поток возобновится под этим блоком).
none Разрешить наличие плавающих элементов с обеих сторон.
inherit Указывает, что значение наследуется от родительского элемента.
